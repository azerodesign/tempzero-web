Documentation

Apa itu Tempzero?

Tempzero adalah layanan email sementara untuk menghindari spam pada inbox utama lu. Gunakan email yang digenerate di sini untuk mendaftar layanan yang mencurigakan.

Cara Penggunaan

1. Klik Copy untuk menyalin alamat email aktif.
2. Gunakan email tersebut untuk mendaftar.
3. Klik Refresh jika balasan belum muncul.
4. Klik New atau Delete untuk mendapatkan alamat baru.

API Reference

Lu bisa nge-fetch email masuk lewat API menggunakan *API Key* yang ada di halaman API Access.

// GET /api/inbox/:email fetch(`https://temp.azero.my.id/api/inbox/${encodeURIComponent('youremail@tempzero.my.id')}`, { headers: { 'Authorization': 'Bearer tz_live_YOUR_API_KEY', 'Content-Type': 'application/json' } }) .then(res => res.json()) .then(data => console.log(data));
